Introdução à Planilhas Eletrônicas

Planilhas eletrônicas são softwares que usam tabelas formadas por grades de linhas e colunas. Ao ser alimentado com informações de escolha do usuário, essas tabelas podem realizar cálculos ou apresentações desses dados de diversas formas.

Exemplos de algumas planilhas eletrônicas disponíveis no mercado:

  1. Planilhas Google;
  2. Numbers;
  3. Quip;
  4. Excel.

Apesar da variedade existente de planilhas eletrônicas, a mais usada é o Excel, fornecida pela empresa Microsoft, disponível entre os programas do Pacote Office.

Sendo o Excel o programa mais usado para esse tipo de objetivo, iremos apresentar aqui um tutorial para a familiarização com o programa, através de conceitos básicos para a criação de uma planilha.

O Excel está disponível para assinantes do pacote Microsoft 365, ele pode ser baixado e instalado no computador ou pode ser usado “online” através de qualquer navegador de internet, iremos usar aqui a versão 2013 do Excel para o tutorial.

Criação De Planilhas

Ao abrir o software e criar uma “pasta de trabalho em branco”, irá aparecer a página inicial com a grade de linhas e colunas vazias, pronto para começar a planilha.

Destacado em cores estão as ferramentas básicas para começar a entender e trabalhar com o Excel:

  1. Amarelo: faixa de opções onde estão disponíveis as principais ferramentas;
  2. Vermelho: linhas da grade dispostas em ordem numérica (1,2,3,4,5...);
  3. Preto: colunas da grade dispostas em ordem alfabética (A, B, C, D, E...);
  4. Verde: célula.

Entrada De Dados

Para começarmos a entender e aplicar conceitos iniciais do Excel, vamos criar uma simples tabela contendo nove linhas e duas colunas, sendo de nomes de pessoas aleatórias e suas respectivas idades, trabalhando assim, a entrada de dados, numéricos (idade) e textos (nomes).

Para inserir os dados basta clicar na célula e digitar a informação requerida.

  1. Preto: tabela;
  2. Vermelho: nas duas colunas da primeira linha foi colocado os "títulos", nomes e idades;
  3. Amarelo: nas linhas da primeira coluna foi colocado os nomes das pessoas
  4. Verde: nas linhas da segunda coluna foi colocado as idades

Criando Fórmulas Básicas

Iremos agora aplicar nessa mesma tabela uma fórmula para que se encontre a média aritmética das idades, lembrando, para se encontrar uma média aritmética de qualquer conjunto de números, basta somar todos os números disponíveis e o resultado, dividir pela quantidade de números desse conjunto.

Para aplicar qualquer fórmula no Excel, o primeiro passo é encontrar uma célula livre e para o Excel entender que estamos começando uma fórmula, devemos usar o sinal de igual (=).

Após o sinal de igual, iremos selecionar as células que serão somadas e o sinal de igual entre elas, agora só apertar o “enter” do teclado e a soma das idades será feita.

  1. Amarelo: fórmula criada dentro da célula selecionada;
  2. Preto: células das idades que serão somadas;
  3. Vermelho: resultado da soma das idades.

Agora vamos dividir o resultado obtido pelo número de idades disponíveis, o resultado desse cálculo será a média aritmética.

  1. Vermelho: fórmula criada dentro da célula para encontrar a média aritmética;
  2. Preto: resultado encontrado através da fórmula, a média aritmética entre as idades

Criando Funções

No Excel existem diversas funções para cálculos mais complexos ou que necessitam de muitos valores, e que ficaria difícil selecionar esses valores um por um, igual feito no último exemplo.

A função MÉDIA é uma função do tipo aritmética, o retorno dela é exatamente a média aritmética entre números selecionados.

Vamos aplicar essa função ao invés da fórmula que criamos no exemplo anterior.

  1. Amarelo: função MÉDIA criada dentro da célula, a função MÉDIA recebe os valores que escolhidos para o cálculo, no caso, célula B2 até a célula B9;
  2. Vermelho: os valores que a função MÉDIA irá receber;
  3. Preto: resultado calculado pela função MÉDIA.

Agora suponhamos que queremos entre as idades, as pessoas que tenham idade igual ou superior que sessenta e um (61) anos, nesse caso, usaremos a função SE.

A função SE é do tipo condicional, sua operação é feita através de uma condição, se essa condição for verdadeira, ela mostrará um resultado, se a condição for falsa, ela mostrará outro resultado diferente.

Função SE recebe o teste lógico, o valor se verdadeiro e o valor se falso.

No caso proposto, utilizaremos duas funções SE, uma dentro da outra.

O primeiro teste lógico será, a idade de “Marie Curie” é maior que 61 anos (B2>61), se esse teste se confirmar, o valor verdadeiro exibirá “Maior que 61 anos de Idade”, se o teste for falso, iremos introduzir outra função SE, o segundo teste lógico dessa função será, a idade de “Marie Curie” é igual a 61 anos (B2=61), se esse segundo teste se confirmar, o valor verdadeiro exibirá “Igual a 61 anos de idade”, se o teste for falso, o valor falso exibirá “Menor que 61 anos de idade”, aplicaremos essa função para todas as idades disponíveis dessa tabela.

  1. Vermelho: resultado da função SE para cada idade disponível na tabela;
  2. Preto: idades disponíveis da tabela.

Criando Gráficos

O Excel além de fazer cálculos complexos com os dados fornecidos, também apresenta esses dados de diversas formas, uma delas é através de gráficos.

Para criarmos um gráfico, iremos usar a tabela dos exemplos anteriores.

Nessa tabela tem duas colunas, nomes de pessoas e suas idades, o gráfico que faremos mostrará as idades.

Existem diversos gráficos, o que usaremos será o de “Colunas Agrupadas”.

Para criarmos esse gráfico, iremos selecionar toda a tabela, na faixa de opções de ferramentas, iremos na opção “INSERIR”, nessa aba iremos selecionar a opção “Inserir Gráficos de Colunas” e por último “Colunas Agrupadas”.

  1. Amarelo: faixa de opções de ferramentas, opção “INSERIR”;
  2. Preto: tabela de dados que o gráfico será feito, tabela selecionada;
  3. Vermelho: gráfico “IDADES”;
