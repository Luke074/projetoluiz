Introdução à Sistema Operacional de código fechado

Os softwares de código fechado, ou Closed Source, são os sistemas amplamente usados pelos fabricantes de programas, onde o acesso aos códigos é restringido, podendo ser acessado ou alterado apenas por pessoas previamente autorizadas, como os programadores.


Modo Texto(linha de comando)

em sistemas operacionais como o Windows, não existem um "modo texto" más existe uma forma parecida, utilizamos o prompt de comando, é uma janela aonde é usada o modo Texto porem mais limitado.

Navegação entre diretórios

É possível fazer a navegação no modo gráfico selecionando o diretório desejado ou pelo modo texto com comandos, por exemplo "cd diretorio1".

Criação de diretórios e arquivos

É um tanto limitada , é focado na criação de diretórios e arquivos pelo modo gráfico sem a necessidade de utilizar comandos, porém é possível ciar no modo gráfico usando o "md arquvo1", também é possível fazer isso no modo gráfico é possível utilizar o Ctrl + shift + N ou com o botão direito, apertar em novo e depois escolhe o que deseja criar.

Exclusão de diretórios e arquivos

Para excluir arquivos é preciso utilizar o comando do Cmd, "del nome.txt" assim o arquivo ou diretório será excluído, isso também pode ser feito pelo modo gráfico selecionando o arquivo e apertando DELETE.

Renomeação de diretórios e arquivos

podemos renomear usando os comandos RENAME ou REN, tambem é possivel renomear pelo modo gráfico utilizando o mouse selecione o arquivo ou diretorio com o botão direito, e depois aperte em renomear.

Movimentação de diretórios e arquivos

Geralmente é utiizado no modo gráfico Ctrl + X para recortar e para colar no local desejado Ctrl + V

Cópia de diretórios e arquivos

Podemos usar o comando Copy para copiar pastas e arquivos no cmd, pelo modo gráfico usamos Ctrl + C para copiar e para colar apertamos Ctrl + V para colar no local desejado.

Edição de arquivos

Infelizmente por padrão não é possível editar arquivos no CMD, sendo assim a única opção o modo Gráfico.

Modo gráfico (Interface gráfica)

Interface Gráfica é um conceito da forma de interação entre o usuário do computador e um programa por meio de uma tela ou representação gráfica, visual, com desenhos, imagens etc.

Área de trabalho

A Área de trabalho (também chamada de desktop), como define a ajuda do sistema, “é a principal área exibida na tela quando você liga o computador e faz logon no Windows”.

Trabalho com janelas

Na interface gráfica foi adotado o Trabalho com janelas aonde é possivel manter um arquivo ou página do navegador aberta, sendo possivel minimizar, fechar ou selecionar para trabalhar com determinada janela de forma rapida e facil.

Configuração do sistema

podemos usar o Msconfig para solucionar problemas de software no Windows e acessar as configurações do sistema.

Ferramentas do sistema

Para acessar as ferramentas, clique em Menu Iniciar, Acessórios e depois em Ferramentas.

Acessórios

É possível achar a pasta acessório no menu iniciar no Windows por exemplo:

  • Lixeira do computador;
  • Windows Movie Maker;
  • Windows Media Player;
  • Windows Explorer;
  • Outlook Express;
  • Internet Explorer;
  • Jogos (como Pinball e Campo Minado);
  • Windows Update;

Menu de ajuda

Escreva ajuda no menu iniciar e selecione o ícone de ajuda logo abrirá uma janela que aparece opções de ajudas para o Windows.

gerenciamento de arquivos

O gerenciador de arquivos do Windows é chamado “Explorer”. Ele tem por função principal visualizar os arquivos que existem no computador. O gerenciador de arquivos pode ser acessado de várias formas, por exemplo escrevendo no menu iniciar.

Gerenciamento de usuários e permissões

no menu iniciar, aperte em Painel de controle, logo em seguida aperte em contas de usuário, nesta página será possível ter gerenciamento de tipo de usuário, mudar o nome etc.

níveis de inicialização e finalização

não temos comandos para controlar os níveis de inicialização e finalização no Windows.

Instalação na máquina virtual

Acesse virtualbox.org, baixe e instale a versão do VirtualBox compatível com o seu sistema operacional, após a instalação de uma virtubox é importante ter uma ISO do Sistema Operacional para a instalação.

Pacotes

existem os pacotes Office, que são pacotes que disponibilizam aplicativos que podem ser importantes para o dia a dia no trabalho e até em casa.

Atualização

A atualização do sistema operacional, disponibilizada pelo próprio fabricante. A frequente atualização do sistema operacional impede que softwares maliciosos se alojem no sistema sem nossa permissão e o que é pior muitas vezes sem nosso conhecimento.
Para estar sempre em dia nas atualizações é necessário ter a licença do Sistema Operacional caso contrário você não receberá as atualizações.

programas

é um conjunto de instruções que descrevem uma tarefa a ser realizada por um computador. O termo pode ser uma referência ao código fonte, escrito em alguma linguagem de programação, ou ao arquivo que contém a forma executável deste código fonte. É possível baixar pela internet programas que irão ajudar no seu dia a dia.
