Introdução à Sistema Operacional de código aberto

SO de código aberto (do inglês open Soure) é o Sistema Operacional com o seu código fonte disponibilizado e licenciado com uma licença de código aberto no qual o direito autoral fornece o direito de estudar, modificar e distribuir o SO de graça para qualquer um e para qualquer finalidade. Um exemplo é o Linux.

Modo Texto(linha de comando)

Para alterar para o modo texto do Linux usamos o comando Ctrl+Alt+F1.

Navegação entre diretórios

com o comando "CD" é possível navegar entre os diretórios, digitando CD pasta01 caso queira voltar para o diretório anterior apertamos CD.. e para voltar pro primeiro diretório apertamos CD~.

Criação de diretórios e arquivos

Com o comando mkdir podemos criar os diretórios escrevendo dessa maneira (mkdir exemplo).

Exclusão de diretórios e arquivos

O comando rm inicialmente é utilizado para apagar um arquivo, porém se utilizado com o parâmetro -r é possível apagar um diretório e todo o seu conteúdo.

Renomeação de diretórios e arquivos

Para renomear podemos usar o comando mv escrevendo dessa forma (mv file1 newnameefile1).

Movimentação de diretórios e arquivos

Comando cp, permite copiar arquivos ou diretórios para outro local escrevendo dessa forma ($ cp /etc/passwd /home/Fabio/) comando mv também pode ser usado para mover arquivos.

Cópia de diretórios e arquivos

é possível usar o comando cp escrevendo dessa forma (cp /home/seusuer/Desktop/ /tmp/)

Edição de arquivos

para editar usamos o VI. Para abrir um arquivo nele, basta digitar vi e o nome do arquivo. Por exemplo, vi index.html.

Modo gráfico (Interface gráfica)

Para ir para modo gráfico no Linux podemos usar o comando CTRL+ALT+F7.

Área de trabalho

A Área de trabalho (também chamada de desktop), é uma forma de usar o Sistema Operacional no modo gráfico. Assim que seu computador é ligado e fazemos o login, aparece a Área de trabalho.

Trabalho com janelas

Na interface gráfica foi adotado o Trabalho com janelas aonde é possível manter um arquivo ou página do navegador aberta, sendo possível minimizar, fechar ou selecionar para trabalhar com determinada janela de forma rápida e fácil.

Configuração do sistema

é possível acessar o menu de Configurações do sistema ou System settings.

Ferramentas do sistema

  • Java Oracle;
  • Webmin;
  • Boot-Repair-Disk;
  • Máquina Virtual;
  • GParted;

Acessórios

no menu do linux é possivel encontrar um icone nomeado de acessorios que vai abrir uma página mostrando diversos acessorios do linux.

Menu de ajuda

Podemos usar o comando man que irá abrir o manual com alguns tópicos para ajudar com suas dúvidas.

gerenciamento de arquivos

a linha de comando é o mais utilizado em questão de gerenciador de arquivos mais para utilizar de modo gráfico é recomendado utilizar o DOLPHIN.

Gerenciamento de usuários e permissões

Gerenciamento de usuários e grupos do Linux. O Linux é um sistema multiusuário, com isso nenhum usuário pode interferir nas atividades do outro, ou seja, não pode alterar arquivos ou configurações de outro usuário. As restrições implantadas no sistema de permissões do Linux são consideradas muito eficientes.

níveis de inicialização e finalização

níveis de inicialização e finalização:
0: De forma direta, esse nível desliga o sistema. Também conhecido como halt.
1, s ou S: Nível monousuário.Utilizado pelo superusuário para realizar atividades básicas para manutenção do sistema (sem acesso à rede ao compartilhamento de arquivos).
2, 3, 4 e 5: Multiusuário.
6: Reinicialização do sistema (reboot).

Pacotes

Um pacote de software ou arquivo de pacote é um software empacotado num formato de arquivo que permite que aplicativos ou bibliotecas sejam distribuídos através de um sistema de gerenciamento de pacotes. O objetivo do empacotamento é permitir, de modo automático instalar, atualizar, configurar e remover programas de computador para o Sistema Operacional. Os formatos de pacote contêm, além de distribuição de software, metadados adicionais, como um arquivo de manifesto, licenças de copyright, diretórios e configurações de instalação e do aplicativo. Também podem conter distribuição de código-fonte, muito comum em distribuições Linux

Atualização

A atualização do sistema operacional, disponibilizada pelo próprio fabricante. A frequente atualização do sistema operacional impede que softwares maliciosos se alojem no sistema sem nossa permissão e o que é pior muitas vezes sem nosso conhecimento.
Para estar sempre em dia nas atualizações é necessário ter a licença do Sistema Operacional caso contrário você não receberá as atualizações.

1 Abra o terminal do Ubuntu, utilizando a tecla "Ctrl + Alt + T" ou pelo Painel Inicial pesquise por "Terminal" e abra o aplicativo.
2 Execute o comando "update-manager -d" no Terminal.
3 No Atualizador de programas, clique no botão "Configurações..." e informe sua senha para abrir o aplicativo "Programas e atualizações".
4 Selecione a aba "Atualizações" no aplicativo Programas e atualizações.
5 Confirme se a opção "Noticar-me de uma nova versão do Ubuntu" está definida como "Para qualquer nova versão". Caso contrário, mude-a.
6 Feche o aplicativo Programas e atualizações e retorne para o Atualizador de programas.
7 No Atualizador de programas, clique no botão "Verificar" parar verificar por novas atualizações.
8 Se houver quaisquer atualizações, use o botão "Instalar agora" para instalá-las e pressione "Verificar" novamente após elas serem instaladas.
9 Uma mensagem vai aparecer informando a disponibilidade de uma nova versão.
10 Clique em "Atualizar".
11 Siga as instruções na tela.
