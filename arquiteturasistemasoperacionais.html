Introdução à Arquitetura dos Sistemas Operacionais

A arquitetura dos sistemas operacionais é uma matéria que estuda o funcionamento dos sistemas operacionais. Um sistema operacional (SO) é uma coleção de programas que inicializam o hardware do computador. Fornece rotinas básicas para controle de dispositivos. Fornece gerência, escalonamento e interação de tarefas. Mantém a integridade de sistema.


Histórico

O Unix é considerado o primeiro sistema Operacional, foi responsável pela criação de muitos outros sistema operacionais, o Unix foi lançado em novembro de 1971, ele era um sistema que utilizava Interface de base texto.

O MS-Dos também foi um sistema Operacional com Interface de base texto, lançado em agosto de 1981, foi responsável pela criação do sistema Operacional Windows. O Windows iria ser um sistema Operacional focado na utilização da interface de modo gráfico, sendo assim com o tempo um dos Sistema Operacional mais usado no mundo superado apenas pelo andoid.

Classificação

Os sistemas operacionais podem ser classificados de formas diferentes, por exemplo, as Características básicas da Arquitetura do Sistema sendo Monolítico, Microkernel ou Híbrido.

Também podemos classificar a Capacidade de execução das tarefas sendo Monotarefa ou Multitarefa, também é possível classificar a Quantidade de usuários que podem operar o sistema classificando-o como Monousuário ou Multiusuário.

Monotarefa

Como o próprio nome implica, o sistema operacional permite que apenas um programa (tarefa) seja executado de cada vez (monoprogramável).

multitarefa

O termo multitarefa refere-se à capacidade que um sistema operacional tem de rodar mais de um programa ao mesmo tempo. Nesse esquema, o sistema operacional mantém uma lista dos processos (ou programas) que estão sendo executados.

Gerenciador de processos

É uma área do sistema operacional onde é possível acompanhar como o seu sistema operacional trabalha com os aplicativos e os processos da GPU. Nessa Aba, ao clicar em um processo com o botão direito, podemos finalizar o processo, finalizar árvore de processos (finaliza tudo relacionado ao processo desejado) e definir a prioridade de execução do processo.

Sistemas de arquivos

Um sistema de arquivos é um conjunto de estruturas lógicas e de rotinas, que permitem ao sistema operacional controlar o acesso ao disco rígido. Diferentes sistemas operacionais usam diferentes sistemas de arquivos.
